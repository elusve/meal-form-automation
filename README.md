# meal-form-automation
automation script written in [Python3](https://en.wikipedia.org/wiki/Python_(programming_language)) with [Selenium](https://www.selenium.dev/)

## Setup
Make sure you have [Python3](https://www.python.org/downloads/), [Selenium4](https://pypi.org/project/selenium/), [chrome](https://www.google.com/intl/en_ca/chrome/), and [chromedriver](https://googlechromelabs.github.io/chrome-for-testing/) installed.
